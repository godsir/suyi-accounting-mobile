## 系统架构对比

### 原有桌面应用特点
- 企业级餐饮/零售管理系统
- 多店铺权限管理
- 10个支付渠道收入管理
- 复杂的财务分析和报表
- Excel批量导入
- 智能文本解析
- 授权许可证管理

### 移动端移植目标
按照原有系统架构，创建功能对等的移动端版本，保持业务逻辑一致性。
